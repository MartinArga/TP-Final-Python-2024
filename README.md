# TP-Final-Python-2024
Proyecto final del curso de Python del ITBA. Año 2024.
## Instalación del proyecto usando Conda

```
conda create -n <env>
conda activate <env>
conda install python==3.10.13
conda install jupyterlab==4.2.5
pip install -r requirements.txt
```
